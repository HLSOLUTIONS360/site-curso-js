Sobre o Curso

Formação direta ao ponto para quem deseja dominar JavaScript na prática, com aulas curtas, atividades mão-na-massa e acompanhamento do projeto final.

Foco em prática
Exemplos reais e desafios semanais.
Metodologia progressiva
Fundamentos → DOM → Lógica → Projeto.
Carga horária estimada
~40 horas, incluindo o módulo bônus.

Grade do Curso

Organização em módulos clara e objetiva:

  • Introdução e ambiente (VS Code, extensões úteis)
  • Variáveis (let, const), tipos e conversões
  • Operadores aritméticos, relacionais e lógicos
  • Strings, template literals e manipulações básicas
  • Boas práticas e organização
  • Seletores, criação/alteração de elementos
  • Classes, estilos, acessibilidade básica
  • Eventos (click, input, submit, teclado)
  • Formulários e validação
  • Feedback visual e microinterações
  • Condicionais (if/else, switch)
  • Laços (for, while, for...of)
  • Funções (declaração, arrow, parâmetros padrão)
  • Arrays (map, filter, reduce) e objetos
  • JSON e datas
  • Briefing, protótipo e cronograma
  • Componentização simples com funções
  • LocalStorage e persistência de dados
  • Refino, testes e publicação

Módulo Bônus — Git & GitHub

Controle de versão na prática, do zero ao push do seu projeto.

  • Instalar Git (Windows, macOS e Linux)
  • Configuração global: nome, e-mail e editor
  • Integrando com o VS Code
  • Chaves SSH e autenticação
  • git init, status, stage e commit
  • Conectar repositório remoto (git remote)
  • git push e git pull
  • .gitignore e README bem feito
  • Branches: criar, trocar, mesclar
  • Estratégia simples de ramificação
  • Pull Requests (visão geral) e revisão
  • Resolvendo conflitos básicos
